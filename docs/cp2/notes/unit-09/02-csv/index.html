Lesson 02 - CSV Files

What is a CSV file?

  • CSV stands for “comma-separated values,” and refers simply to a text file in which data entries are separated by commas!
    • similarly, you might hear TSV (tab)

Importing / Exporting CSV with Java

  • Use String methods and File I/O
  • Importing
    • Use a BufferedReader
    • split the string and pass parameters to a Command
  • Exporting
    • Use a BufferedWriter
    • join an array
